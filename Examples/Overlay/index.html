jQuery.elementOverlay

Welcome to the demo page of elementOverlay, a jQuery overlay script which can put an overlay over any HTML element.

Setup

Will show you how to configure the script and it's required styles

CSS styles

You need to start by applying styling to these items: 

.element-overlay-contents {  color: red; text-align: center; font-size: 22pt; }
.element-overlay-bk {}
.element-overlay-contents
The text/spinner etc are put in a div with this class.
.element-overlay-bk
Used for the actual overlay.

You do typically just need to change the color for the above defined style.

Invoking the overlay

You simply call the plugin as: 


$('#css-list').elementOverlay();
$('#css-list').elementOverlay({title: '<img src="spinner.gif" />'});
$(window).elementOverlay();

Options

These are the available options

overlay
HTML or ID (starting with #) for the overlay
spinner
Container used to display information over the overlay. Either HTML or and element ID (starting with #).
title
Text to show over the overlay

Methods

Methods which can be invoked on an overlay

show
Show a previously hidden overlay
hide
hide an overlay
destroy
Hide and remove everything from the DOM tree

Example

$('#css-list').elementOverlay('hide');

Code

The code is available at github.com. You only need the jquery.griffin.elementOverlay.js script (and the above mentioned CSS styles).
