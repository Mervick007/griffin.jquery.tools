jQuery.griffin.Tour

Welcome to the demo page of griffin.tour, a jQuery Tour script. The script is used to guide the user through your UI and the features that your site contain.

Click on this demo link to show what's it about

Setup

Will show you how to configure the script and it's required styles

CSS styles

You need to start by applying styling to a couple of items: 

.griffin-tour-spotlight { position:relative; z-index: 9999; background: #CCEEFF; }
#griffin-tour-container { display: none; background: #CCEEFF; padding: 20px; position: absolute; z-index: 9999; }
#griffin-tour-overlay { display: none; z-index: 9998;}
.griffin-tour-spotlight
Added to the item which is described
#griffin-tour-container
Container that the guide is added to. The container contains the Next/Prev buttons
#griffin-tour-overlay
Overlay used to gray out everything else. Will get the jQueryUI overlay theme

You do typically just need to change the colors for the above defined styles

Creating the guide

The guide is created by using one DIV per item to describe. The DIV can contain anything you like.

Example guide 

<div style="display:none" id="the-tour">
    <div>
        <h1>Welcome</h1>
        <p>Welcome to the tour of this page!</p>
        <p>Click "Next" to start the tour</p>
    </div>
    <div data-for="#setup">
        <h1>The Setup</h1>
        <p>Click on this link to learn how to setup and configure the plugin</p>
    </div>
</div>
  • data-for attribute is used to point at the element to describe
  • No data-for attribute centers the guide (try scrolling)

Invoking the tour

You simply call the plugin as: 


$('#tourToStart').griffinTour();

Customize

You can customize the container to use a nicer tour layout. Add a new div somewhere and assign it a ID:

<div id="my-custom-container" class="griffin-tour-container" style="display: block">
    <div class="contents">
    </div>
    <div class="footer">
        <a href="#" class="previous ui-button"><< Previous</a>
        <a href="#" class="quit ui-button">Quit</a>
        <a href="#" class="next ui-button">Next >></a>
    </div>
</div>

It's important that the links get the previous / quit / next CSS classes and that there are an element with the contents class.

Next up is to add the custom styling:

#my-custom-container { display: none; background: #CCEEFF; padding: 20px; position: absolute; z-index: 9999; padding: 0px;}
#my-custom-container div p { padding-left: 10px; padding-right: 10px }
#my-custom-container a.ui-button { text-decoration: none;  }
#my-custom-container h1 { margin: 0px; border-bottom: 1px solid #cccccc; background: white; padding: 10px;}
#my-custom-container .footer { border-top: 1px solid #cccccc; background: white; margin: 0px; padding: 10px; }

Now just init (click on the link) the tour with the id: 


$('#tourToStart').griffinTour({container: '#my-custom-container'});

Options

These are the available options

effect
One of the jQueryUI effects (like 'highlight' which is the default)
container
The container that the tour items will be placed in
overlay
A custom overlay
